# rendercv_resume

The professional resume of Damon Myers, generated with RenderCV.

## Setup

Install rendercv with:

``` sh
uv tool install "rendercv[full]"
```

## Render

Rendered resumes are ignored in .gitignore.

To render a resume, run:

``` sh
rendercv render damon_myers.yaml
```

Rendered files will be written to `rendercv_output`.
